# Supply Management (การจัดการสต็อกวัสดุสิ้นเปลือง)

คู่มือการควบคุมสต็อกสินค้าและของใช้สิ้นเปลืองสำหรับหอพัก คอนโด และโฮสเทล (เช่น กระดาษทิชชู่, สบู่, แชมพู, แปรงสีฟัน, ผ้าเช็ดตัว):

## 1. จุดสั่งซื้อขั้นต่ำ (Reorder Point / Minimum Stock)
- สินค้าแต่ละรายการจะมีค่า `minimum_stock` กำหนดไว้ (เช่น ทิชชู่ = 5 ม้วน, สบู่เหลว = 3 แกลลอน)
- หากยอดคงคลังลดต่ำกว่าขั้นต่ำนี้ ระบบจะต้องแจ้งเตือน `SUPPLY_LOW` ทันที

## 2. ขั้นตอนการอัปเดตสต็อก
- เมื่อแม่บ้านนำของไปใช้ในห้องพัก ให้ทำการบันทึกตัดสต็อกด้วย `update_supply_quantity(item_id, quantity, 'decrease')`
- เมื่อสั่งของเข้ามาเพิ่ม ให้จดบันทึกยอดรับด้วย `update_supply_quantity(item_id, quantity, 'increase')`

## 3. การสั่งซื้ออัตโนมัติ (Auto-Reorder)
- หากรายการใดถูกตั้งค่า `auto_reorder = true` เมื่อสินค้าต่ำกว่าจุดวิกฤต AI Agent จะทำหน้าที่ร่างใบสั่งซื้อ (Purchase Order) ผ่านคู่ค้าที่บันทึกไว้ในระบบทันที
- ส่งใบเสนอสั่งซื้อไปให้ผู้จัดการกดอนุมัติการโอนจ่ายเงิน
- สำหรับรายการอื่นๆ AI จะสร้างรายการแจ้งเตือนเตือนความจำในรายงานสรุปประจำสัปดาห์
