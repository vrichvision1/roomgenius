# Lease Management (การจัดการสัญญาเช่า)

แนวปฏิบัติในการจัดการสัญญาเช่าตั้งแต่วันแรกจนถึงวันย้ายออก:

## 1. การทำสัญญาเช่าใหม่ (New Leases)
- สัญญาเช่าประกอบด้วยข้อมูล: ชื่อผู้เช่า, ห้องพัก, อัตราค่าเช่า, เงินมัดจำ, วันเริ่มสัญญา, วันสิ้นสุดสัญญา
- อัตราค่าเช่าสามารถกำหนดเป็น รายวัน (Daily), รายเดือน (Monthly) หรือรายปี (Yearly)
- ก่อนบันทึกสัญญา ผู้ดูแลต้องทำการบันทึกเงินมัดจำแยกต่างหาก (Deposit Payment)

## 2. การต่ออายุสัญญา (Lease Renewal)
- สำหรับผู้เช่าที่ต้องการพักอาศัยต่อ AI Agent จะมีบทบาทในการเช็คสัญญาเช่าที่กำลังจะหมดอายุล่วงหน้า **30-60 วัน**
- ดึงข้อมูลผ่าน `get_expiring_leases()`
- AI จะติดต่อผู้ดูแลและผู้เช่าเสนอโปรโมชั่นหรือสอบถามความประสงค์ในการต่อสัญญา หากประสงค์จะต่อสัญญา AI จะสร้างเอกสารต่อสัญญาและอัปเดตผ่านเครื่องมือ `renew_lease()`

## 3. การยกเลิกหรือสิ้นสุดสัญญา (Lease Termination)
- เมื่อผู้เช่าย้ายออก หรือผิดนโยบายจนต้องยกเลิกสัญญา ให้ใช้เครื่องมือ `terminate_lease()`
- ในวันย้ายออก ผู้ดูแลต้องทำการตรวจสอบความเสียหายของห้องพัก และคำนวณเงินมัดจำที่จะคืน (หลังจากหักค่าน้ำค่าไฟงวดสุดท้ายและค่าความเสียหายแล้ว)
- AI จะช่วยคำนวณยอดเงินที่ต้องคืนหรือเรียกเก็บเพิ่มตามประวัติการจดค่าใช้จ่าย
